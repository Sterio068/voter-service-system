2. Windows 安裝步驟

2.1 取得安裝程式

向系統管理員索取最新版安裝程式，檔案名稱格式如：選民服務系統-Setup-1.0.0.exe

2.2 執行安裝

  1. 以滑鼠右鍵點擊安裝程式，選擇「以系統管理員身分執行」 若系統彈出 UAC 視窗（使用者帳戶控制），點擊「是」允許繼續。
  2. 選擇安裝語言 預設為「繁體中文」，點擊「確定」。
  3. 選擇安裝目錄 預設安裝於 C:\Program Files\選民服務系統\。若需更改位置，點擊「瀏覽」選擇其他目錄，再點擊「下一步」。
  4. 選擇是否建立捷徑 勾選「建立桌面捷徑」及「建立開始功能表捷徑」（建議保持預設），點擊「安裝」。
  5. 等待安裝完成 安裝程式會自動解壓縮並安裝所有必要元件，過程約 30 秒。
  6. 點擊「完成」 勾選「立即啟動選民服務系統」即可直接開啟，進入首次設定精靈。
Windows Defender SmartScreen 警告 首次執行時 Windows 可能顯示「Windows 已保護您的電腦」警告。點擊「更多資訊」→「仍要執行」即可繼續。此為正常現象，非安全問題。

2.3 防毒軟體設定（如需要）

部分防毒軟體可能誤判本系統。如遇阻擋，請將以下路徑加入防毒白名單：

  • C:\Program Files\選民服務系統\（安裝目錄）
  • C:\Users\[使用者名稱]\AppData\Roaming\選民服務系統\（資料目錄）

3. macOS 安裝步驟

3.1 取得安裝程式

向系統管理員索取適合您 Mac 的版本：

  • Intel Mac：選民服務系統-1.0.0-x64.dmg
  • Apple Silicon（M1/M2/M3）：選民服務系統-1.0.0-arm64.dmg

3.2 安裝流程

  1. 雙擊 .dmg 檔案開啟磁碟映像 系統會掛載一個虛擬磁碟並開啟安裝視窗。
  2. 將「選民服務系統」圖示拖曳至「應用程式」資料夾 按照視窗提示，將應用程式拖曳至右側的 Applications 捷徑。
  3. 退出磁碟映像 拖曳完成後，在 Finder 側欄對磁碟映像點擊退出按鈕。
  4. 首次開啟 在「應用程式」資料夾找到「選民服務系統」，按住 Control 鍵並點擊，選擇「開啟」。系統會詢問確認，點擊「開啟」即可。
Gatekeeper 說明 首次開啟時 macOS 可能顯示「無法打開，因為開發者無法驗證」。請前往「系統設定」→「隱私權與安全性」→「安全性」，找到相關訊息，點擊「仍要開啟」。此為 macOS 安全機制，屬正常情況。
Apple Silicon 注意事項 M 系列 Mac 若出現效能問題，請確認下載的是 arm64 版本。Intel 版本可在 M 晶片上透過 Rosetta 2 執行，但效能較差。

4. 首次設定精靈

首次啟動時，系統會引導您完成資料儲存位置的設定。這個步驟非常重要，決定了所有資料的儲存位置。

4.1 選擇儲存位置

精靈會顯示以下對話框，提供兩個選項：

選項說明適用情況
使用本機預設位置 儲存在此台電腦的應用程式資料夾（AppData） 單機使用、無需跨電腦共用
選擇自訂位置... 可選擇本機任意資料夾，或輸入 NAS / 網路磁碟路徑 多台電腦共用、備份至 NAS
預設路徑位置
  • Windows：C:\Users\[使用者]\AppData\Roaming\選民服務系統\
  • macOS：~/Library/Application Support/選民服務系統/

4.2 自訂位置注意事項

  • 選擇的資料夾必須有讀寫權限，系統會自動測試寫入
  • 若寫入測試失敗，會自動改回本機預設位置
  • 系統會自動在選擇的路徑下建立 uploads/ 和 backups/ 子目錄

4.3 完成設定

設定完成後，系統顯示確認訊息並開始正常運作。日後若需變更資料位置，可從應用程式選單 → 「系統」→「變更資料位置...」重新設定。

設定完成後 資料位置僅需設定一次。重新啟動後系統會自動使用已設定的路徑，不會再次顯示精靈。

5. 區域網路 / NAS 資料共用設定

若您的辦公室有 NAS（網路附接儲存）或共享資料夾，可以將資料庫儲存於其上，實現多台電腦共用同一份資料。

5.1 事前準備

  • 確認 NAS 或共享資料夾已掛載並可正常存取
  • 確認您的帳戶對該資料夾有完整的讀寫權限
  • 建議使用有線網路，避免 Wi-Fi 連線不穩造成資料損毀
  • 若使用 NAS，確認 SMB 協定已啟用

5.2 Windows 設定 NAS 路徑

  1. 先掛載網路磁碟機 開啟「檔案總管」→「本機」→ 右鍵「連接網路磁碟機」，輸入 NAS 路徑，例如：\\192.168.1.100\公務文件\選民服務，指派磁碟機代號（如 Z:）。
  2. 啟動選民服務系統時選擇「自訂位置」 在首次設定精靈中點選「選擇自訂位置...」，在對話框中導覽至剛才掛載的網路磁碟機，選擇目標資料夾。
  3. 確認寫入測試通過 系統會自動測試寫入權限，通過後顯示成功訊息。

5.3 macOS 設定 NAS 路徑

  1. 連接 SMB 共享 Finder 選單 → 「前往」→「連接伺服器」，輸入 smb://192.168.1.100/公務文件，輸入帳號密碼後掛載。
  2. 選擇自訂位置 在首次設定精靈中選擇「選擇自訂位置...」，導覽至 /Volumes/公務文件/ 下的目標資料夾。
重要：開機自動掛載 若電腦重新開機後 NAS 未自動掛載，系統啟動時會偵測到路徑失效，並提示您重新設定。建議在系統設定中設定開機自動掛載網路磁碟機，以避免此情況。

6. 多台電腦共用資料庫

本系統支援多台電腦共用同一個資料庫檔案，適合辦公室多人共同維護選民資料的需求。

6.1 架構說明

共用模式的架構如下：

  • 所有電腦的選民服務系統均指向 NAS 上的同一個 voter-service.db 檔案
  • 系統使用 SQLite WAL 模式，支援多個讀取端同時存取
  • 同一時間只能有一個寫入端（系統內建鎖定機制保護資料完整性）

6.2 各電腦設定步驟

  1. 第一台電腦：建立初始資料庫 安裝並首次設定時，選擇 NAS 上的共用資料夾作為資料儲存位置，完成初始化。
  2. 其他電腦：指向相同資料夾 在其他電腦安裝系統後，首次設定時同樣選擇「自訂位置」，指向 NAS 上的相同資料夾。系統會自動偵測到已有資料庫並直接使用，不會覆蓋現有資料。

6.3 注意事項與限制

項目說明
同時寫入多人同時操作（寫入）時，系統會自動等待並重試，若等待超過 15 秒會顯示錯誤提示
網路中斷操作過程中若網路中斷，正在執行的操作會失敗並回滾，已完成的操作不受影響
NAS 版本建議使用支援 SMBv2 以上的 NAS（QNAP、Synology 等主流品牌均支援）
同時連線人數建議不超過 10 台電腦同時連線，以確保穩定性
嚴禁以下行為
  • 直接複製 .db 檔案作為備份時，請確保系統未在運作
  • 不可使用雲端同步服務（如 OneDrive、Google Drive）同步 .db 檔案，這會造成資料損毀
  • 不可將資料庫放在 USB 隨身碟上共用，延遲過高會造成鎖定問題

7. 常見問題 FAQ

安裝後雙擊圖示沒有反應，或閃一下就關掉？

請嘗試以系統管理員身分執行，或檢查防毒軟體是否阻擋程式啟動。也可查看系統事件檢視器的應用程式記錄找尋錯誤訊息。

啟動時出現「伺服器無法啟動」錯誤？

可能是 8080 連接埠被其他程式占用。請關閉可能使用 8080 埠的程式（如其他 Web 服務），或在工作管理員中結束相關行程後重試。

資料路徑設定後，下次開機找不到資料？

最常見原因是 NAS 未自動掛載。請確認網路磁碟機設定為開機自動連線，或在開啟系統前手動掛載網路磁碟機。

如何查看目前的資料儲存位置？

開啟系統後，點擊選單 →「系統」→「變更資料位置...」，對話框中會顯示目前的資料路徑。

Windows 防毒軟體一直刪除程式檔案？

請將安裝目錄（通常為 C:\Program Files\選民服務系統\）加入防毒軟體的排除清單。若使用 Windows Defender，可在「Windows 安全性」→「病毒與威脅防護」→「管理設定」→「排除項目」中新增。

可以在同一台電腦安裝多個版本嗎？

不建議。每次安裝新版本前，請先解除安裝舊版本，資料庫不會被刪除。

macOS 更新後系統無法啟動？

macOS 系統更新有時會重設安全性設定。請再次至「系統設定」→「隱私權與安全性」允許應用程式執行，或重新下載安裝程式。

資料庫檔案有多大？效能如何？

一般辦公室規模（約 5,000 位選民、2,000 件陳情）資料庫約 10-50 MB，查詢速度極快（毫秒級）。資料庫本身支援至數 GB，不影響使用。

手機可以用嗎？

可以。確保手機與主機在同一 WiFi 後，在手機瀏覽器輸入主機 IP（可在「設定」→「網路資訊」查看），即可使用行動版介面。

如何將備份自動存到外接硬碟？

在系統「設定」→「資料庫備份」→「變更目錄」，選擇外接硬碟的路徑。之後所有手動備份與自動備份都會存至該目錄。請確保外接硬碟在備份時間點已連線。

上傳附件後，檔案存在哪裡？

附件存於資料目錄下的 uploads/attachments/ 資料夾。備份資料庫時，建議同時手動備份此資料夾，或將整個資料目錄一起備份。

8. 解除安裝

注意：解除安裝前請先備份資料 解除安裝程式本身不會刪除您的資料庫，但若要完全清除所有資料，需要手動刪除資料目錄。

Windows 解除安裝

  1. 前往「設定」→「應用程式」→「已安裝的應用程式」
  2. 找到「選民服務系統」，點擊右側選單 → 「解除安裝」
  3. 依照解除安裝精靈完成程序
  4. （選擇性）手動刪除資料目錄：%APPDATA%\選民服務系統\

macOS 解除安裝

  1. 在 Finder 中開啟「應用程式」資料夾
  2. 將「選民服務系統」拖曳至「垃圾桶」
  3. （選擇性）刪除資料目錄：~/Library/Application Support/選民服務系統/
