## 一、專案概述

### 1.1 專案目標

開發一套**本地單機版選民服務管理系統**，支援 Windows 與 macOS 一鍵安裝，提供議員服務處、里長辦公室、候選人團隊及地方黨部等政治服務單位使用。系統涵蓋選民資料管理、陳情追蹤、選務行政、公文管理與行程排程等核心功能。

### 1.2 設計原則

| 原則 | 說明 |
|------|------|
| **離線優先** | 所有資料存放本地，不依賴雲端服務，確保資料自主權 |
| **一鍵安裝** | 使用者無需安裝額外環境（Node.js、資料庫等），安裝即用 |
| **跨平台** | 同一套程式碼產出 Windows (.exe/.msi) 與 macOS (.dmg) 安裝檔 |
| **多人協作** | 支援區域網路內 3-10 人同時存取，一台主機部署、其餘透過瀏覽器連線 |
| **資料安全** | 登入密碼加密儲存、資料庫本地存取、操作紀錄完整留存 |

### 1.3 使用情境

```
┌─────────────────────────────────────────────────┐
```

## 二、技術架構

### 2.1 技術選型建議

| 層級 | 技術 | 選用理由 |
|------|------|----------|
| **桌面殼** | Electron | 生態成熟、跨平台穩定、打包工具完善（electron-builder） |
| **前端** | React + TypeScript + Ant Design | 元件豐富、表單處理強、中文化完善、適合後台管理系統 |
| **後端** | Node.js + Fastify | 輕量高效、與 Electron 同語言、易於打包 |
| **資料庫** | SQLite（better-sqlite3） | 零配置、嵌入式、WAL 模式支援多讀者、備份簡單 |
| **ORM** | Drizzle ORM | 型別安全、輕量、支援 SQLite |
| **報表列印** | Puppeteer（內建 Chromium）/ react-to-print | 支援 PDF 匯出與列印 |
| **安裝打包** | electron-builder | 產出 .exe/.msi (Win) 與 .dmg (Mac) |

### 2.3 部署架構

| 模式 | 說明 |
|------|------|
| **單機模式** | Electron 視窗內直接使用，伺服器僅監聽 localhost |
| **區網模式** | 主機開放區網 IP，其他電腦透過瀏覽器 `http://192.168.x.x:8080` 連入 |

## 三、功能模組規劃

### 3.1 系統管理

#### 3.1.1 帳號維護

| 功能 | 說明 |
|------|------|
| 帳號建立 | 管理員新增使用者帳號（帳號、姓名、密碼、角色） |
| 角色權限 | 預設角色：管理員、主管、助理、志工（僅檢視） |
| 權限矩陣 | 依模組設定：檢視 / 新增 / 編輯 / 刪除 / 匯出 |
| 密碼管理 | bcrypt 加密儲存、強制首次登入修改密碼、密碼強度檢查 |
| 帳號停用 | 可停用帳號而非刪除，保留歷史操作紀錄完整性 |
| 登入管理 | JWT Token 認證、閒置 30 分鐘自動登出、登入失敗鎖定 |

**角色權限矩陣範例：**

| 功能模組 | 管理員 | 主管 | 助理 | 志工 |
|----------|--------|------|------|------|
| 系統管理 | 全部 | 檢視 | - | - |
| 帳號維護 | 全部 | - | - | - |
| 操作紀錄 | 全部 | 檢視 | - | - |
| 陳情管理 | 全部 | 全部 | 新增/編輯 | 檢視 |
| 選民資料 | 全部 | 全部 | 新增/編輯 | 檢視 |
| 團體資料 | 全部 | 全部 | 新增/編輯 | 檢視 |
| 公文管理 | 全部 | 全部 | 新增/編輯 | 檢視 |
| 行程管理 | 全部 | 全部 | 新增/編輯 | 檢視 |
| 名單列印 | 全部 | 全部 | 全部 | - |

#### 3.1.2 操作紀錄（稽核日誌）

| 欄位 | 說明 |
|------|------|
| 操作時間 | 精確到秒的時間戳記 |
| 操作人員 | 帳號名稱與角色 |
| 操作類型 | 登入/登出/新增/修改/刪除/匯出/列印/查詢 |
| 操作模組 | 所屬功能模組名稱 |
| 操作對象 | 被操作的資料識別（如選民姓名、陳情案號） |
| 操作內容 | 變更前/後內容摘要（修改時記錄差異） |
| IP 位址 | 操作者的連線 IP |

- 操作紀錄僅供檢視與匯出，**不可刪除或修改**
- 支援依時間範圍、人員、操作類型篩選
- 支援匯出 CSV/Excel 報表

#### 3.1.3 系統設定

| 項目 | 說明 |
|------|------|
| 基本資訊 | 服務處名稱、負責人、聯絡資訊、Logo 上傳 |
| 區域設定 | 服務區域行政區劃設定（縣市/鄉鎮區/村里） |
| 類別管理 | 陳情類別、選民標籤、團體類別的自訂管理 |
| 備份還原 | 手動/排程自動備份資料庫、一鍵還原 |
| 備份目錄 | 可自訂備份儲存路徑（如外接硬碟、NAS），不需重啟即生效 |
| 網路設定 | 區網模式開關、埠號設定 |
| 系統版本 | 設定頁顯示目前安裝版本號 |

### 3.2 陳情管理

#### 3.2.1 陳情案件

| 欄位 | 說明 |
|------|------|
| 案件編號 | 系統自動產生（格式：年份-流水號，如 2026-000001） |
| 陳情日期 | 案件受理日期 |
| 陳情人 | 連結選民資料（可快速查詢或新建） |
| 陳情方式 | 電話/親訪/書信/LINE/電子郵件/轉介/其他 |
| 陳情類別 | 可自訂分類（道路、水溝、路燈、噪音、社福、就業、法律、交通、環保、都市計畫、其他） |
| 陳情內容 | 詳細文字描述 |
| 處理區域 | 行政區/村里/路段地址 |
| 急迫程度 | 一般/急件/特急 |
| 處理狀態 | 待處理 → 處理中 → 已轉介 → 已回覆 → 已結案 → 已歸檔 |
| 承辦人員 | 指派負責的助理 |
| 附件 | PDF、照片上傳（支援多檔，單檔最大 20MB）；圖片可全螢幕預覽，PDF 在新分頁開啟 |

#### 3.2.2 陳情處理流程

```
陳情受理 → 案件分類 → 指派承辦 → 處理/轉介
                                      │
                    ┌─────────────────┤
                    │                 │
                 自行處理          轉介公部門
                    │                 │
                    │            發文追蹤
                    │                 │
                    └────────┬────────┘
                             │
                         回覆陳情人
                             │
                          結案歸檔
```

#### 3.2.3 陳情追蹤

| 功能 | 說明 |
|------|------|
| 處理紀錄 | 每次處理動作（電話、會勘、發文等）逐筆記錄 |
| 提醒機制 | 逾期未處理自動標示警示（可設定天數） |
| 轉介追蹤 | 記錄轉介對象（公所、局處等）與追蹤進度 |
| 滿意度 | 結案時記錄陳情人滿意度（滿意/尚可/不滿意） |
| 統計報表 | 依類別/區域/狀態/時間產出統計圖表 |

## 四、資料庫設計

### 4.1 核心資料表

```sql
-- 使用者帳號
CREATE TABLE users (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    username    TEXT NOT NULL UNIQUE,
    password    TEXT NOT NULL,          -- bcrypt 加密
    name        TEXT NOT NULL,
```

## 七、資料安全與備份

### 7.1 安全措施

| 項目 | 實作方式 |
|------|----------|
| 密碼儲存 | bcrypt（cost factor 12） |
| 敏感欄位 | 身分證字號等欄位使用 AES-256 加密存取 |
| 認證機制 | JWT Token + HttpOnly Cookie |
| 閒置登出 | 30 分鐘無操作自動登出（可設定） |
| 登入鎖定 | 連續 5 次失敗鎖定 15 分鐘 |
| 操作紀錄 | 全面記錄、不可刪改 |

### 7.2 備份策略

| 備份類型 | 說明 |
|----------|------|
| 手動備份 | 系統設定中一鍵備份，產出帶時間戳記的 .db 檔案 |
| 自動備份 | 可設定每日/每週自動備份，保留最近 N 份 |
| 備份位置 | 預設在安裝目錄下 `/backups`，可自訂路徑（如外接硬碟） |
| 備份還原 | 選擇備份檔案一鍵還原（還原前自動備份當前版本） |
| 匯出匯入 | 支援將全部或部分資料匯出為加密壓縮檔，可在另一台電腦匯入 |
